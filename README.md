# CodeAlpha Hangman Game

## Description
A text-based Hangman Game developed using Python as part of the CodeAlpha Python Programming Internship.

## Features
- Random word selection
- Maximum 6 incorrect attempts
- Input validation
- Console-based gameplay

## Technologies Used
- Python 3

## How to Run

```bash
python hangman.py
```
